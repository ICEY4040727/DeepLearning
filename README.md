# DeepLearning

Deep Learning notebooks and experiments.

## Run the notebooks in the browser (free, public repo)

- Binder (JupyterLab):

  Use this badge to open the repository on Binder (public repos only):

  <a href="https://mybinder.org/v2/gh/ICEY4040727/DeepLearning/HEAD"><img src="https://mybinder.org/badge_logo.svg"></a>

- Gitpod (VS Code-like browser workspace):

  Open in Gitpod (public repos have free usage quota):

  https://gitpod.io/#https://github.com/ICEY4040727/DeepLearning

- GitHub Pages:

  The repository is configured to build HTML from notebooks into the docs/ folder and publish via GitHub Pages. To enable:
  1. In the repository Settings -> Pages, set Source to the main branch and /docs folder.
  2. After the workflow runs, the site will be available at https://ICEY4040727.github.io/DeepLearning/

- Pre-commit nbstripout

  Developers should install pre-commit locally:

  ```
  pip install pre-commit
  pre-commit install
  ```
